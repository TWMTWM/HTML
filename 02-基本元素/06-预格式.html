一、没有使用预格式情况：

1、没有放在p标签里

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello World"); } }

2、放在p标签里

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello World"); } }

二、使用预格式：

		public class HelloWorld {
		 
		    public static void main(String[] args) {
		        System.out.println("Hello World");
		    }
		}
